# Dashboard di coinvolgimento web

Il dashboard del coinvolgimento web fornisce visibilità sul modo in cui i visitatori web interagiscono con i contenuti chiave. Segmenta i dati tra i settori e le aree geografiche degli account per aiutarti a comprendere le tendenze di coinvolgimento. Utilizza questa dashboard per supportare il processo decisionale strategico presentando modelli di comportamento web che informano la strategia dei contenuti e il targeting degli account.

Per accedere al _dashboard di Web Engagement_, seleziona l&#39;elemento **[!UICONTROL Dashboard]** nell&#39;area di navigazione a sinistra. Quindi seleziona la scheda **[!UICONTROL Coinvolgimento Web]** nella parte superiore della pagina.

![Dashboard di coinvolgimento Web](./assets/web-engagement-dashboard.png){width="700" zoomable="yes"}

## Filtrare i dati

Fai clic sull&#39;icona _Filtro_ ( ![Icona Filtro](../assets/do-not-localize/icon-filter.svg) ) in alto a sinistra per filtrare i dati visualizzati utilizzando uno dei seguenti attributi:

* **[!UICONTROL Area account]** - Filtra i dati in base a una o più aree geografiche selezionate associate agli account.
* **[!UICONTROL Settore account]** - Filtra i dati in base a una o più classificazioni di settore selezionate associate agli account.
* **[!UICONTROL Intervallo date]** - Filtra i dati in base all&#39;intervallo date selezionato. L&#39;intervallo predefinito è la data corrente.

![Filtra i dati visualizzati per attributi](./assets/web-engagement-dashboard-filters-dialog.png){width="500"}

Selezionare tutti i valori per ogni attributo che si desidera utilizzare per filtrare i dati e fare clic su **[!UICONTROL Applica]**.

## [!UICONTROL Visualizzazioni pagine principali] {#top-page-views}

>[!CONTEXTUALHELP]
>id="ajo-b2b_web_engagement_top_page_views"
>title="Visualizzazioni pagina principali"
>abstract="Le pagine visualizzate più di frequente sul sito web, che consentono di identificare quale contenuto genera più traffico."

Questa tabella mostra le 10 pagine web più visualizzate, per aiutarti a identificare quale contenuto risuona di più con i visitatori. I dati includono:

| Colonna | Descrizione |
| ------ | ----------- |
| Nome pagina | Nome o titolo della pagina web. |
| Visualizzazioni totali | Il numero totale di volte in cui la pagina è stata visualizzata. |
| Visitatori noti (%) | La percentuale di visualizzazioni di pagina attribuite a visitatori noti (identificati). |
| Visitatori sconosciuti (%) | La percentuale di visualizzazioni di pagina attribuite a visitatori sconosciuti (anonimi). |

## [!UICONTROL Visualizzazioni pagina per area account] {#page-views-by-region}

Questa visualizzazione mostra i conteggi dei visitatori segmentati per regione dell’account. ## Coinvolgi i dati

Per interagire con i dati, utilizza il menu _Altro_ (**...**) in alto a destra di ciascun grafico e scegli **[!UICONTROL Visualizza altro]** per visualizzare dati e approfondimenti estesi.

Il pop-up visualizzato include un grafico e una tabella che mostrano la suddivisione dei dati.

Per scaricare i dati, fai clic su **[!UICONTROL Scarica CSV]** in alto a destra nella tabella dei dati.

![Tabella visualizzazioni pagine principali Visualizza altro popup - Fai clic su Scarica CSV](./assets/web-engagement-dashboard-top-page-views-view-more.png){width="700" zoomable="yes"}
